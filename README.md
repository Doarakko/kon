# kon

![crates](https://img.shields.io/crates/v/kon.svg) ![Licence](https://img.shields.io/github/license/Doarakko/kon)

Display fox image in the terminal.

## Install

cargo

```sh
cargo install kon
```

Homebrew

```sh
brew tap Doarakko/kon
brew install kon
```

## Usage

```sh
kon
```
